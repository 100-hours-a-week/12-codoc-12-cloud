1. VS Code에서 Terraform 코드 작성
2. GitHub에 push (선택)
3. 로컬 터미널에서 아래 코드 실행
- terraform init
    - 이 디렉토리를 “Terraform 프로젝트”로 초기화
    - AWS provider 다운로드, .terraform/ 폴더 생성
- terraform fmt
    - .tf 파일 코드 정렬 (자동 포맷, 스타일 통일용)
- terraform validate
    - 문법/참조 오류 체크
- terraform plan
    - “이 코드로 apply하면 AWS에 뭐가 생기고/바뀌는지” 미리 보여줌
	- 아직 실제 변경 ❌
- terraform apply
    - 실제로 AWS에 리소스 생성
    - VPC / Subnet / EC2 / SG 다 만들어짐
4. AWS 확인

- 결론
terraform init
terraform fmt
terraform validate
terraform plan
terraform apply
